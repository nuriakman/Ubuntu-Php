## Karşılaştırma Operatörleri

| Operatör | Açıklama |
|----|----|
| \== | Eşittir. |
| === | Aynıdır, Denkse. |
| \!= | Eşit değildir. |
| \<> | Eşit değildir. |
| \!== | Farklıdır. |
| \< | Küçüktür. |
| \> | Büyüktür. |
| \<= | Küçük yada eşittir. |
| \>= | Büyük yada eşittir. |
| <=> | Mekik (Php 7 ve üstü).|

*İpucu 1: Sonuçlar true(1) yada false(0) döner.*

*İpucu 2: Denkse de değişkenlerin hem değerleri hem de türleri eşit olacak.*
