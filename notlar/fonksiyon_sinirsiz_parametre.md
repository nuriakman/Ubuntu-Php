# FONKSİYONLARDA SINIRSIZ PARAMETRE GÖNDERMEK

Bu tarz fonksiyonlar genellikle "*root*" sistemlerde kullanılırlar.

Örneğin;

```php
  // Kaç parametre gelmiş onu buluruz.
  func_get_args();

  // Parametrelerin listesini dizi olarak döndürebiliriz.
  func_get_args();

  // İndex numarası vererek parametrenin değerine ulaşırız.
  func_get_arg();
```
